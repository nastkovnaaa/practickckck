# Сервисный центр "Новое дело" - Информационный ресурс

## Описание проекта
Разработка концепции и прототипа информационного ресурса для автоматизации сервисного центра по ремонту бытовой электроники.

## Цель работы
Создание основы для разработки нового информационного ресурса, включая анализ бизнес-процессов, проектирование интерфейса, тестирование прототипа и применение инструментов командной разработки.

## Структура репозитория

| Папка | Содержание |
|-------|------------|
| `/docs/analysis` | Аналитические документы, BPMN диаграммы (AS-IS, TO-BE) |
| `/docs/reports` | Отчеты по юзабилити-тестированию |
| `/prototypes` | HTML/CSS/JS прототипы экранов |
| `/diagrams` | Сайтмэп и другие схемы |

## Содержимое прототипов

| Файл | Описание |
|------|----------|
| screen1-home.html | Главная страница |
| screen2-order.html | Форма создания заявки |
| screen3-tracking.html | Отслеживание статуса ремонта |
| screen4-admin.html | Панель администратора |

## Использованные технологии
- HTML5, CSS3
- Git / GitHub
- BPMN 2.0
- Figma

## Этапы работы

1. **Анализ бизнес-процессов** — построение моделей AS-IS и TO-BE в нотации BPMN 2.0
2. **Проектирование и прототипирование** — создание кликабельного прототипа в Figma
3. **Тестирование** — юзабилити-тестирование с выявлением проблем и рекомендаций
4. **Git** — управление версиями, ветки, pull request
