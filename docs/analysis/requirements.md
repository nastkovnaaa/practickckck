# Требования к информационному ресурсу

## Функциональные требования
1. Создание заявки на ремонт
2. Отслеживание статуса ремонта
3. Управление заявками для администратора
4. Ведение учета запчастей
5. Автоматические уведомления клиентов

## Нефункциональные требования
1. Интуитивно понятный интерфейс
2. Адаптация под мобильные устройства
3. Время отклика не более 3 секунд
